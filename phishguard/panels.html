PhishGuard AI
v1.0.0 — Phishing Detection Tool

PhishGuard AI is a full-stack phishing detection system that analyses emails and URLs to identify phishing attacks using machine learning, NLP, and real-time threat intelligence. It combines rule-based heuristics, a trained Random Forest classifier, and live blacklist lookups to deliver accurate, explainable results.

// DETECTION FEATURES
✓ SPF / DKIM / DMARC verification
✓ NLP body text analysis
✓ URL feature extraction (15 signals)
✓ WHOIS domain age check
✓ DNS anomaly detection
✓ PhishTank blacklist lookup
✓ Google Safe Browsing API
✓ ML classifier (Random Forest)
// TECH STACK
Python 3.11 Flask REST API React.js scikit-learn spaCy / NLTK BeautifulSoup PostgreSQL Redis + Celery Docker JWT Auth
